# Medical Appointments System - Landing Page

A modern and responsive landing page for a medical appointments system. This basic version showcases the frontend interface built with Angular, featuring a clean design and smooth user experience without backend functionality or admin dashboard.

## üåü Features

- **Responsive Design**: Fully responsive layout that adapts seamlessly to desktop, tablet, and mobile devices.
- **Modern UI**: Clean and professional interface with smooth animations and transitions.
- **Custom Styling**: Carefully crafted design system with custom CSS variables for consistent theming.
- **Server-Side Rendering**: Angular SSR enabled for improved performance and SEO optimization.
- **Fast Loading**: Optimized with Vite for quick build times and efficient bundling.
- **Accessible**: Built following web accessibility best practices for inclusive user experience.
- **Smooth Navigation**: Intuitive routing and navigation with Angular Router.
